# Der XSLT-Stylesheet tei2mets
Start: 28.07.2017

Der XSLT-Stylesheet tei2mets basiert auf dem tei2mets.xsl Style, der sich innerhalb des an der SUB Göttingen entwickelten METS Merger (siehe https://github.com/subugoe/mets-merger) befindet.

Dieser Stylessheet wurde vorallem für die Transformation von Texten des Deutsches Textarchives (http://www.deutschestextarchiv.de) angepasst. Vorallem für die Auswertung des TEI-HEADER wurden neue XSL-Transformationen geschrieben.
